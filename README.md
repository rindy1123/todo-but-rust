# Just TODO app but all written in Rust

This is just a normal TODO web app but everything is written in Rust with simple API and simple UI(No CSS, sorry.)

## Getting Started

### Dependencies

- Docker
- Rust 1.x.x
- Rocket 0.5.0-rc.3
- Yew 0.20.0
- SurrealDB 1.0.0-beta.9

### How to run

```
docker compose up -d
```

That's it!
